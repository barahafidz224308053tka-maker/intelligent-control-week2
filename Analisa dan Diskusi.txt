Analisa dan Diskusi
Praktikum ini kita melakuakn percobaan untuk Klasifikasi Objek dengan Scikit-learn dengan menggunakan metode algoritma KKN (algoritma K-Nearest Neighbors ) dan juga menggunakan metode algortima SVM (Support Vector Machine), Langkah pertama yang dilakukan adalah membuat clone repositori dari GitHub ke Visual Studio Code, Langkat kedua adalah melakukan install lbrary yang dibutuhkan, Langkah ketiga adalah membuat dataset warna yang akan digunakan pada deteksi warna nantinya, Langkah ketiga adalah membuat program pada masing masing metode deteksi, baik program deteksi KKN maupun SVM, jangan lupa memhubungkan kode program dengan dataset yang telah dibuat sebelumnya, setelah itu dapat dilakukannya running pada setipa program yang dibuat pada masing masing metode. Pada deteksi warna menggunakan KKN output dari deteksi meliputi : label warna, bounding box, nilai accuracy dan Confidence Score, sedangkan pada metode SVM didapatkan output meliputi : 2 titik deteksi, bounding box, label warna, Confidence Score dan nilai accuracy. dalam melakukan percobaan menggunakan kedua metode tersebut, didapatkan tingkat accuracy sebesar 65-100% dan dengan nilai confidence sebesar 85-100%, kedua nilai tersebut bergantung juga pada kualitas kamera dan intensitas cahaya di lingkungan tersebut
